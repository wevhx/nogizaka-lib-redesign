NOGIZAKA LIB

这是一个简洁明了地显示乃木坂46的信息的网页应用。

前往应用

支持语言

ENGLISH日本語简体中文

功能：音乐

CD 列表

CD 列表界面会显示出至今为止已经发布的所有 CD（单曲和专辑）卡片的网格列表。

轻触或点击 CD 卡片会移动到相应的 CD 列表界面。

CD

CD 界面会显示出这张 CD 的详细信息。

轻触或者点击 CD 封面可以切换至这张 CD 的其他封面图像。

收录于该 CD 中的歌曲一览也会显示在这个界面中。轻触或点击歌曲卡片会移动到相应的歌曲界面。

歌曲

歌曲界面会显示出这首歌曲的详细信息。

封面图像为这首歌曲所在的 CD 的封面图像。

这首歌曲的表演成员的一览也会显示在这个界面中。轻触或点击成员卡片会移动到相应的成员界面。

在表演成员一览的下方将会显示该歌曲的创作人员一览（例如，作词、作曲）。

功能：成员

成员列表

成员列表界面会显示出一个所有成员（1期生到4期生、毕业生）的网格列表。

轻触或点击成员卡片会移动到相应的成员列表界面。

成员

成员界面会显示出这名成员的详细信息。

成员照片下的圆点表示这名成员的荧光棒颜色。

成员的姓名、网站、生日、出生地等的信息也会显示在这个界面中。

站位记录会表示出成员加入至今的单曲标题曲的站位记录。

站位次数会表示出成员经历过的站位的次数。

官方照片为成员加入至今的官方照片的一览。

功能：搜索

搜索

在画面上方的输入区域中输入 CD 或歌曲的标题，或者成员的名字。

搜索结果会在输入区域的下方显示出来。
